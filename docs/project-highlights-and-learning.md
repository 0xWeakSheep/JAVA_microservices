# 项目亮点与学习收获

本项目的亮点主要体现在它不是一个只完成基础页面展示的简单课程作业，而是尽量按照真实项目的思路进行了完整设计和实现。在业务上，系统围绕电影浏览、电影详情查看、用户注册登录、发表评论、管理员维护电影信息、用户权限管理和日志查看等功能展开，形成了比较完整的电影评论平台闭环；在技术上，项目采用了前后端分离架构，前端使用 `Next.js + React + TypeScript` 构建页面与交互，后端使用 `Spring Boot` 提供 REST API，并结合 `Spring Data MongoDB` 完成数据持久化，使用 `Spring Cloud Gateway` 作为统一入口实现路由转发，同时通过 `JWT + 拦截器 + 注解权限控制` 完成接口鉴权和管理员权限管理，还结合 `AOP` 实现操作日志记录，配合初始化数据、序列号生成和 `PM2` 进程管理，使整个项目不仅具备功能实现能力，也具备一定的工程化和可部署性，这些都使项目相比单纯的增删改查更有完整性和展示价值。

通过这个项目，我最大的收获是对完整项目开发流程和 Spring Boot 工程化开发方式有了更加系统的理解。以前更多是停留在单个知识点学习层面，而这次是从数据库设计、后端接口编写、前端页面开发、前后端联调，到最后构建、部署、重启和文档整理，完整经历了一次项目落地过程；在这个过程中，我进一步掌握了 Spring Boot 中常见的分层结构设计、依赖注入、配置类、拦截器、AOP、MongoDB 数据访问等开发方法，也理解了认证与授权的区别，学会了如何通过 JWT 管理用户登录状态和接口访问权限；同时，在前端部分我也学习了如何封装接口请求、维护页面状态、保存 token、组织组件化页面结构，并在不断处理编译错误、类型错误、进程管理和部署问题的过程中提升了自己的排错能力和工程意识。整体来看，这个项目让我不只是学会了“如何写功能”，更重要的是开始理解一个项目怎样从代码实现走向结构清晰、可以运行、可以展示、可以维护的完整系统。
